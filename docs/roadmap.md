# Roadmap

## Stage 1: CSV Risk API

- Portfolio CSV input
- Price CSV input
- Risk summary endpoint
- Risk report endpoint
- Basic methodology docs
- Tests

## Stage 2: Richer Risk Reports

- Better report structure
- More complete diagnostics
- Optional HTML report export
- Stronger validation notes

## Stage 3: Scenario Engine

- User-defined shocks
- Asset-class shocks
- Currency shocks
- Volatility and liquidity scenario notes

## Stage 4: Data Adapters

- Cleaner normalized schema
- Adapters for different market data sources
- CSV remains the first stable input format

## Stage 5: Optional Dashboard or Hosted Demo

- Lightweight dashboard if the API layer is solid
- Hosted demo with sample data only
- No investment advice and no trading functionality
